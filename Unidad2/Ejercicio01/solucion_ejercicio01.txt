Ejercicio 01 — Verificando tu distribución

El objetivo de este ejercicio es confirmar que tu distribución de LaTeX
(MiKTeX, TeX Live o MacTeX) está correctamente instalada, sin necesidad
de compilar documentos ni usar editores todavía.

--------------------------------------------
1. Windows (MiKTeX)
--------------------------------------------
• Abre “MiKTeX Console”.
• Si la consola abre sin errores, la instalación es correcta.
• En la sección “Updates”, confirma que aparece la lista de paquetes.
• En la sección “Packages”, verifica que puedes ver el catálogo (no necesitas instalar nada aún).

--------------------------------------------
2. macOS (MacTeX)
--------------------------------------------
• Abre la aplicación “TeX Live Utility”.
• Si la lista de paquetes carga correctamente, tu instalación está activa.
• También puedes abrir TeXShop para confirmar que el sistema detecta la distribución (no compiles nada todavía).

--------------------------------------------
3. Linux (TeX Live)
--------------------------------------------
• Abre la aplicación que uses para revisar paquetes del sistema
  (por ejemplo, “Software”, Synaptic o la terminal).
• Confirma que “texlive” o “texlive-full” aparece instalado.
• Si deseas, revisa la carpeta:
/usr/local/texlive
para verificar que la estructura existe.

--------------------------------------------
4. Comprobación ligera de paquetes (sin compilar)
--------------------------------------------
Para confirmar que la distribución es funcional, revisa:

• En MiKTeX:
  La opción “Install missing packages on-the-fly” está disponible.

• En TeX Live:
  El programa “TeX Live Utility” (macOS) o “tlmgr” (Linux) aparece en el sistema.

• En MacTeX:
  La carpeta:
/Library/TeX/texbin
se encuentra presente.

Si puedes realizar estas comprobaciones, tu distribución está instalada
correctamente. No necesitas compilar ni abrir un editor en este punto.
