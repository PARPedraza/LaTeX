# LaTeX para Universitarios

## Crea tesis, artículos y presentaciones como un profesional

Repositorio oficial de recursos complementarios del libro **LaTeX para Universitarios**.

Este espacio reúne ejercicios, soluciones, plantillas y proyectos integradores desarrollados a lo largo del libro con el objetivo de fortalecer las competencias de escritura académica, documentación técnica y publicación científica utilizando LaTeX.

---

## Contenido

### Unidades del libro

- Unidad 1. Introducción a LaTeX
- Unidad 2. Software y flujo de trabajo
- Unidad 3. Estructura y organización de documentos
- Unidad 4. Texto, listas, tablas, figuras y algoritmos
- Unidad 5. Matemáticas en LaTeX
- Unidad 6. Gestión de bibliografía y citas
- Unidad 7. Diseño y personalización de documentos
- Unidad 8. Creación de presentaciones con Beamer
- Unidad 9. Trabajo en plataformas online
- Unidad 10. Temas avanzados

---

## Laboratorio de proyectos integradores

Los siguientes proyectos permiten aplicar de forma práctica los conocimientos adquiridos en el libro:

1. Documento académico básico
2. Reporte técnico profesional
3. Manual de práctica de laboratorio
4. Currículum Vitae académico
5. Artículo científico breve
6. Gestión bibliográfica con BibTeX
7. Presentación académica con Beamer
8. Póster científico
9. Plantilla de tesis universitaria
10. Libro académico completo

---

## Objetivo del repositorio

Este repositorio tiene como finalidad proporcionar material complementario descargable para los lectores del libro, incluyendo:

- Archivos fuente en LaTeX.
- Plantillas listas para modificar.
- Ejercicios resueltos.
- Recursos académicos.
- Ejemplos reproducibles.

---

## Requisitos recomendados

- TeX Live
- MiKTeX
- Overleaf
- Visual Studio Code + LaTeX Workshop

---

## Cómo utilizar los proyectos

1. Descargue el proyecto deseado.
2. Descomprima el archivo ZIP.
3. Abra el archivo principal (`main.tex`).
4. Compile el documento.
5. Personalice el contenido según sus necesidades.
